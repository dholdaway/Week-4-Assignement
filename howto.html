How I Made The Site

the site uses bootstrap, the main image gallery uses javascript to change attributes of page elements

the images are all from marvel and the text is from wikipedia.

Clicking on the small images in the character gallery changes 1 attribute and 2 html elements.

$(".crop-img").click(function(){

// goes to the attribute of the big image and replaces the image source ( url)
    
             $("#bigImage").attr('src',
        
         // replaces the attribute source of big image with the attribute source of the clicked element
     		
		$(this).attr('src'));
            
     // replaces the html content of mainname with the name attribute of the clicked element
            
     $("#mainname").html($(this).attr('name') );
            
     // replaces the html content of the mainsection with the details attribute ( attributes are between the tag)
            
     $("#mainsection").html($(this).attr('details') );
            
});
